Esta aplicación web permite al usuario visualizar datos de COVID-19 actualizados para los diferentes estados de EE.UU. El usuario puede seleccionar un estado desde un menú desplegable y, al presionar un botón, se realiza una solicitud a la API de COVID Tracking Project para obtener los datos clave: casos confirmados, hospitalizaciones, recuperaciones y muertes, los cuales se muestran en tarjetas de manera clara y estructurada.

Es una herramienta sencilla que ofrece información rápida sobre la situación de COVID-19 en cualquier estado de EE.UU., presentando los datos de manera fácil de leer para el usuario.
